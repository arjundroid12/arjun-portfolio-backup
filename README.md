# 🎨 Premium Portfolio — Arjun Vashishtha

Award-quality dark purple portfolio with 3D background, scroll animations, custom cursor, animated counters, and template-matched layout.

## ✨ Features
- 3D background (Three.js wireframe shapes + stars)
- Custom cursor follower (purple dot)
- Scroll progress bar
- Animated stat counters (0 → target on scroll)
- Staggered hero entrance animation
- Project modal with spring physics
- Testimonials carousel
- Contact form with purple borders
- Solid purple footer
- Template-matched layout (EDITION 2026 badges, "FIND my..." headings, numbered cards)

## 🛠️ Tech
- Next.js 16 + TypeScript
- Three.js + React Three Fiber
- Framer Motion + GSAP
- Tailwind CSS 4 + shadcn/ui
- Inter font

## 📦 Run
```bash
npm install
npm run dev
```
